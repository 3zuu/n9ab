# 🎭 مين النصاب؟ — ألعاب جماعية عربية

لعبة الإمبوستر العربية بثلاث أوضاع للعب مع نظام نقاط و5 جولات.

## 📁 الملفات

| الملف | الوصف |
|-------|-------|
| `index.html` | الموقع الرئيسي وواجهة اللعبة |
| `admin.html` | لوحة التحكم الكاملة |
| `vercel.json` | إعدادات النشر على Vercel |

## 🚀 النشر على Vercel

1. ارفع الملفات على GitHub
2. اذهب إلى [vercel.com/new](https://vercel.com/new)
3. اختر المستودع → Deploy
4. كل تحديث على GitHub يتنشر تلقائياً ✅

## 🎮 الألعاب

- **نسخة الأسئلة** — الكل يجاوب سؤالاً واحداً إلا النصاب
- **نسخة التصنيف** — رتّب أفضل 3، النصاب ما يعرف التفاصيل
- **نسخة الرسم** — ارسم الكلمة، النصاب عنده كلمة مختلفة

## ⚙️ لوحة التحكم

افتح `yoursite.com/admin` للوصول للوحة التحكم حيث يمكنك:
- تغيير اسم الموقع والنصوص
- تعديل أيقونات وأسماء الألعاب
- إضافة/حذف/تعديل الأسئلة
- تعديل الفوتر
- تصدير index.html المحدّث
