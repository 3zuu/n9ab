└── vercel.json <-- إعدادات Vercel
📋 خطوات الرفع
الخطوة 1 — إنشاء مستودع GitHub

اذهب إلى github.com/new وأنشئ مستودع جديد (Public). ثم ارفع الملفات الثلاثة.

الخطوة 2 — ربط Vercel

اذهب إلى vercel.com/new واختر المستودع → Deploy. سيعطيك رابط مباشر.

الخطوة 3 — التحديث التلقائي

كل مرة ترفع ملفاً جديداً على GitHub → Vercel يتحدث تلقائياً خلال ثوانٍ!

⚡ تدفق العمل اليومي
1. عدّل من لوحة التحكم
2. احفظ → حمّل index.html
3. ارفع على GitHub
4. Vercel يتحدث تلقائياً ✅

🔥 إعداد Firebase للمزامنة الحقيقية

اجعل أصدقاءك يلعبون معك من أي مكان في العالم

⚡ Firebase مجاني 100% للألعاب

Firebase Realtime Database من Google مجاني تماماً للاستخدام الشخصي والمتوسط. يتيح لك مزامنة اللاعبين والإجابات والتصويت في الوقت الفعلي بدون سيرفر.

الخطوات (5 دقائق فقط)
1️⃣ إنشاء مشروع Firebase

اذهب إلى console.firebase.google.com ← اضغط "إضافة مشروع" ← اختر اسم ← تابع.

2️⃣ تفعيل Realtime Database

من القائمة الجانبية ← Build ← Realtime Database ← Create Database ← اختر منطقة ← ابدأ بـ "test mode".

3️⃣ إضافة تطبيق Web

من Project Overview ← اضغط أيقونة </> ← اسجّل التطبيق ← ستظهر لك إعدادات الاتصال (firebaseConfig).

4️⃣ أدخل الإعدادات هنا 👇
إعدادات Firebase الخاصة بك
📋 قواعد الأمان (Realtime Database Rules)

انسخ هذه القواعد في Firebase Console ← Realtime Database ← Rules

{
  "rules": {
    "rooms": {
      "$roomId": {
        ".read": true,
        ".write": true
      }
    }
  }
}
