🚀 دليل الرفع على Vercel

خطوات نشر موقعك

📁 الملفات المطلوبة
your-repo/
├── index.html   ← الموقع الرئيسي
├── admin.html   ← لوحة التحكم
└── vercel.json  ← إعدادات Vercel
الخطوات
1️⃣ أنشئ مستودع GitHub

اذهب إلى github.com/new → مستودع Public → ارفع الملفات الثلاثة.

2️⃣ اربط Vercel

اذهب إلى vercel.com/new → اختر المستودع → Deploy.

3️⃣ التحديث (كل مرة)
1. عدّل من لوحة التحكم → احفظ
2. حمّل index.html الجديد
3. ارفعه على GitHub
4. Vercel يتحدث تلقائياً ✅
📄 vercel.json
{"version":2,"routes":[{"src":"/","dest":"/index.html"},{"src":"/admin","dest":"/admin.html"}]}
